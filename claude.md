# 个人全局配置

## 回答偏好
- 回复使用中文
- 代码修改前先简要说明修改思路，不要直接给出代码
- 遇到有多种实现方案时，列出选项让我选择，而不是直接选一种

## 通用约定
- 提交信息使用英文类型+中文描述，格式：`type(scope): 描述`
- 新文件开头不加版权注释
- 优先使用原生 API，避免引入不必要的依赖
- 代码注释必须使用中英双语

## 安全习惯
- 修改认证相关代码前主动提示我注意安全影响
- 不要在代码注释或日志中输出任何密钥或 token

## 技术栈偏好（2026 插件清单配置）
- 包管理器：优先使用 pnpm，禁止使用 npm
- TypeScript：使用 type 而非 interface
- 测试框架：使用 Vitest 而非 Jest
- 代码风格：2空格缩进，分号结尾，单引号
- 验证：所有功能必须有测试，提交前必须运行测试
