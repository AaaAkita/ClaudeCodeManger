# Documentation Generator Skill

## 描述
自动生成项目文档，包括 README、API 文档、代码注释等。

## 触发条件
当用户要求以下操作时触发：
- "生成文档"
- "写 README"
- "生成 API 文档"
- "文档化这段代码"
- "create documentation"

## 支持的文档类型

### 1. README.md
项目根目录说明文档，包含：
- 项目简介
- 安装说明
- 使用示例
- 贡献指南
- 许可证信息

### 2. API 文档
- 函数/方法说明
- 参数列表
- 返回值说明
- 使用示例

### 3. 代码注释
- JSDoc / Javadoc 风格
- Python docstring
- 内联注释

### 4. 架构文档
- 系统架构图
- 数据流说明
- 模块关系

## 工作流程

### 1. 分析项目结构
- 识别项目类型（Python、JavaScript、Java 等）
- 分析代码结构
- 提取关键信息

### 2. 生成文档
- 根据项目类型选择合适的文档模板
- 提取代码中的注释和类型信息
- 生成示例代码

### 3. 格式化输出
- 使用 Markdown 格式
- 添加适当的标题和章节
- 插入代码块和表格

## 输出标准

### README 结构
```markdown
# 项目名称

## 简介
简要描述项目功能

## 功能特性
- 特性 1
- 特性 2

## 安装
\`\`\`bash
npm install 包名
\`\`\`

## 使用示例
\`\`\`javascript
// 代码示例
\`\`\`

## API 文档
链接到详细文档

## 贡献
如何贡献代码

## 许可证
MIT / Apache 等
```

## 注意事项
- 保持文档简洁清晰
- 提供实际可运行的示例
- 及时更新文档以反映代码变更
- 使用清晰的中文或英文（根据项目语言）
