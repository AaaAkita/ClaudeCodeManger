# Code Review Skill

## 描述
这个 Skill 帮助 Claude 进行高质量的代码审查，提供结构化的反馈和改进建议。

## 触发条件
当用户要求以下操作时触发：
- "审查这段代码"
- "code review"
- "检查代码质量"
- "review my code"

## 工作流程

### 1. 代码分析
- 检查代码风格和格式
- 识别潜在的 bug 和错误
- 评估代码可读性和可维护性
- 检查性能问题

### 2. 安全性检查
- 查找常见的安全漏洞
- 检查输入验证
- 评估敏感数据处理

### 3. 最佳实践
- 验证是否遵循语言特定的最佳实践
- 检查设计模式的使用
- 评估代码复用性

### 4. 输出格式
使用以下结构输出审查结果：

```markdown
## 代码审查报告

### ✅ 优点
- 列出代码的优点

### ⚠️ 需要改进
- 列出需要改进的地方

### 🐛 潜在问题
- 列出潜在的 bug 或问题

### 🔒 安全建议
- 安全相关的建议

### 📚 最佳实践建议
- 代码改进建议

### 💡 重构建议
- 具体的重构方案（如有必要）
```

## 参考标准
- 遵循 SOLID 原则
- 关注代码可读性
- 优先考虑可维护性
- 注重性能和安全性
